# Sail

A gem to bring into your Rails application settings to be used as knobs and gauges for controlling behavior. 

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sail'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sail
```
